# Astrology vale Baba

A simple astrology-inspired web application built using HTML, CSS, and JavaScript. Users enter their personal details, and the application generates entertaining insights based on their date of birth.

---

## Features

* Displays a zodiac sign based on the birth month
* Generates a compliment based on the birth date
* Shows a humorous victim card compliment
* Provides a random recommendation
* Generates a fun prediction
* Responsive and user-friendly interface

---

## Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla)

## Getting Started

### Clone the Repository


### Run the Project

1. Open the project folder.
2. Open `index.html` in your preferred web browser.

No additional installation or dependencies are required.

## How It Works

The user enters:

* First Name
* Surname
* Birth Date
* Birth Month
* Birth Year

The application then generates:

* Zodiac Sign
* Compliment
* Victim Card Compliment
* Recommendation
* Prediction

The results are generated using predefined JavaScript arrays and simple indexing logic.
